# sound_pad

# 사운드 패드란
사운드 패드라는 제품을 어플로 만들어본 프로젝트이다. 유튜브에서 자주 사용되는 브금을 각각 버튼 1개로 제작하였으며, 녹음, 배속 기능, 정지 등이 가능한 어플이다.  
안드로이드스튜디오에 익숙해지기 위해 연습(?)삼아 만든 어플이며, 플레이스토어에 업로드되었지만, 다른 기기에서 소리가 제대로 발생하지 않는 문제가 생겨 플레이스토에서 내린 상태이다.
